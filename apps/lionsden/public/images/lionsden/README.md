# Lions Den images

Add these files here (they are referenced by the app):

- **logo.png** – Logo shown at top of hero
- **hero.jpg** – Main hero image (e.g. Jordan / family photo)
- **q1.jpg** – Question 1 image
- **q2.jpg** – Question 2 image
- **q3.jpg** – Question 3 image
- **q4.jpg** – Question 4 image

You can use `.png`, `.webp`, or `.jpg`; if you use different extensions, update the paths in `src/apps/lionsden/config.ts` and `src/apps/lionsden/questions.ts`.
